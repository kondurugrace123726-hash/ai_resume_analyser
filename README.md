# ai_resume_analyser
# 🤖 AI Resume Analyzer

A Python project that analyzes resume text, detects technical skills, and provides improvement suggestions.

## 🚀 Features

- Detects skills from resume
- Calculates resume score
- Shows missing keywords
- Gives suggestions
